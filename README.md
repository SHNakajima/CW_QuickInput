# ChatWork Quick Input

## Description

チャットワーク用Chrome Extension。

[Chromeウェブストア]()

## Features

チャットワーク（ChatWork）のチャットで、定型文を素早く入力するための拡張機能です。
設定した定型文を"ボタン押下"or"キーボードショートカット"で入力することができます。

詳しくは[Chromeウェブストア]()をご確認ください。
